# Taxonomies API (Public)

## GET /api/taxonomies/categories
Returns a lean list of categories for filters and chips.

**Response 200**
```json
[
  { "id": 3, "name": "Elderly", "slug": "elderly" }
]
```

## GET /api/taxonomies/tags
Returns a lean list of tags.

**Response 200**
```json
[
  { "id": 7, "name": "Weekend", "slug": "weekend" }
]
```
